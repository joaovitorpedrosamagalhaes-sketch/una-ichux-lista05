# una-ichux-lista05
As heurísticas de usabilidade aplicadas no código têm como objetivo melhorar a experiência do usuário durante a interação com o sistema.

A Heurística 01 – Visibilidade do status do sistema é observada quando o programa informa ao usuário o que está acontecendo, como nas mensagens “Conectando ao Banco Central” e “Calculando taxas”. Isso é importante para que o usuário não fique confuso ou pense que o sistema travou.

A Heurística 05 – Prevenção de erros aparece no uso do bloco try-catch, que evita que o programa encerre de forma inesperada quando o usuário digita um valor inválido. Além disso, é exibida uma mensagem de erro clara, orientando o usuário a inserir apenas números.

A Heurística 08 – Estética e design minimalista está presente na organização visual da interface, com uso de cores, separadores e formatação do texto. Isso torna a leitura mais fácil e deixa o sistema mais agradável e intuitivo.

Dessa forma, o código demonstra preocupação com a usabilidade, tornando a interação mais clara, segura e eficiente para o usuário.
